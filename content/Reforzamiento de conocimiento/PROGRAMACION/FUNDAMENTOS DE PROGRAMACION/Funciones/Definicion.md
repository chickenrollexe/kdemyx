Conjunto de instrucciones ya predefinida que se puede reutilizar para una tarea especifica.
## Funcionalidades
- **Reutilización:** 
El código se escribe una vez y se utiliza múltiples veces en diferentes partes del programa.
- **Modularidad:** 
Divide programas complejos en piezas más pequeñas, organizadas y manejables.
- **Abstracción:** 
Oculta la complejidad de la implementación, el usuario solo necesita saber qué hace la función y qué parámetros necesita, no cómo funciona internamente.
### Usos Comunes:
- **Cálculos Matemáticos:** 
Funciones predefinidas como calcular la raíz cuadrada, elevar a una potencia o generar números aleatorios.
- **Validación de Datos:**
Comprobar si un correo electrónico tiene el formato correcto o si un formulario está completo antes de enviarlo.
- **Manipulación de Cadenas:**
Transformar texto, como convertir minúsculas a mayúsculas o cortar cadenas de caracteres.
- **Gestión de Eventos:** 
Ejecutar acciones al hacer clic en un botón, como `onClick()` en desarrollo web.
- **Conectividad a Bases de Datos:**
Funciones que realizan consultas (queries) para leer o escribir información en una base de datos
#### Como sirven
1. Le das un nombre
2. Parámetros
*Por ejemplo:*
Hacer una receta

```
comida(ingredientes){
  bloque de codigo
  return resultado
}
```
1. Crear función para sumar

```
function sum (a, b){

resultado = a+b

return resultado
}
```
2. Y luego para llamar a esta función y utilizarla:

```
total = sum (1, 2)
```
