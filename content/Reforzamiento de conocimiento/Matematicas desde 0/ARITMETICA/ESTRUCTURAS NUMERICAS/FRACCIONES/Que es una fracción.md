Una fracción es una expresión matemática que representa una o varias partes iguales en las que se ha dividido una unidad o un total.

Se pueden representar visualmente mediante graficas.

Las fracciones son consideradas una estructura numérica fundamental porque representan una relación lógica entre partes y un todo, permitiendo expresar cantidades que no son números enteros. Funcionan como un sistema organizado para dividir, medir y comparar, formando parte del conjunto de los números racionales (Q)
