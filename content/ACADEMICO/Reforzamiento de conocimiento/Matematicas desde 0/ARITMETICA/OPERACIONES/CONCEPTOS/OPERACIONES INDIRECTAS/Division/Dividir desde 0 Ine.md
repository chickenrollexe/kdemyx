# Divisiones inexactas

![[IMG-20260505223431002.png]]
Cuantas veces cabe el 7 en 1?
- no se puede, por lo tanto se usa 12, 12 es mas grande así que se usa de referencia para la tabla de multiplicar.

Cuantas veces cabe 7 en 12?
- se busca el numero mas cercano a 12
- 7x1= 7
- Si se usara 7x2= 14 entonces se pasaría del 12 así que no sirve.

Se pone 7 de cociente encima de 2 porque recordar que se hizo 12.
Se pone el 7 debajo y se hace la respectiva resta.
Pero al momento de restar 2-7, 2 es mas pequeño por lo tanto se le pide prestado 1 al numero de la izquierda del dividendo, en este caso como es 1, el 1 se convierte en 0, ahora el 2 se convierte en 12 (1 se combino con 2).

Ahora 12-7= 5

Después se baja el 0 del 120 quedando 50, entonces se busca el numero mas cercano a 50 que seria 7x7= 49, quedando como cociente 17 porque se subió 7.

Después se hace la resta 50-49, pero como 0 es mas pequeño que 9 se usa lo de anteriormente
- se le pide prestado 1 al del lado que es 5-1=4 y el 9 se convierte en 10.
- 10-9= 1, se baja 1, después el 5 que se convirtió en 4 es 4-4= 0
- quedando como residuo 1 y como no quieres que quede como residuo algo diferente de 0 entonces.

Como no hay ningún numero o 0 que bajar entonces pones un "0" a la derecha de tu dividendo quedando como en este caso 1200, bajas el 0 al 1 quedando 10.

Haces el mismo procedimiento:
- 7x7= 7 el mas cercano a 10
- 10-7: 0-7 = 1 se vuelve 0 porque se presta a 0
- 0 quedando como 10, 10-7= 3.

Puedes añadir los 0 que quieras pero esta división es inexacta es recomendable dejarla hasta ahí.
- Se añadió otro 0 al 3 quedando 30 y como se queda 2 en el residuo de la resta y no 0 como se espera, pues jamás se terminara la división.
