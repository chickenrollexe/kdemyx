Variables que pueden almacenar muchos valores en una sola variable.
**Por ejemplo:**
*si necesitas una lista de compras*
No crearías cada variable para cada compra.
1. Solo crearías una lista bajo solo un nombre.
2. Y una vez creada puedes acceder a ella mediante posiciones
## Propiedades
Las posiciones siempre se empiezan desde 0.
o sea: el primer elemento será 0 en vez de 1.
0- Lunes
1- Martes
2- Miércoles
3- Jueves
4- Viernes
5- Sábado
6- Domingo
