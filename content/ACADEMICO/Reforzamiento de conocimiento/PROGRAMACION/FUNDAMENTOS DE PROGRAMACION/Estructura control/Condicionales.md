# Que son?
"SI hago esto pasara esto"
acción = consecuencia
## Cuales son
if = si se cumple la condición
    siempre acepta condicional

else = si no entonces
    no acepta condicional porque ningún caso se ha cumplido

else if = si no si
    múltiples condiciones
### Importancia:
Tomar distintos caminos por lo tanto tener distintas funcionalidades o acciones.
#### Ejemplo
Si (Vaso de agua)
(Tomar agua)
*Ahora con else*
Si(Vaso de agua)
(Tomar agua)
else
(No tomar agua)
*Ahora con else if*
Si(Vaso de agua)
(Tomar agua)
else if
(No tomar agua)
else
(Tirar vaso)
    Tiene que ser else if ya que si pusieras if else ya crearías otra condición aparte.
